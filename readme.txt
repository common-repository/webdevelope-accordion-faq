== Description ==
Uses custom post types and taxonomies to manage an FAQ section for your site. Includes a set of custom taxonomies to organize, and shortcode options for different display configurations.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the 'webdevelope-accordion-faq' folder to the `/wp-content/plugins/` directory or install via the WP admin panel
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it.

== Frequently Asked Questions ==

= What does this do? =

It uses the custom post type feature to create a dedicated FAQ section in your WordPress site, including categories and tags exclusive to them.
